# v0.1.0 Release Body

## Title

`v0.1.0 - Persistence Engine (Base Regime)`

## Body

Initial public release of HAOS Genesis.

HAOS Genesis is a path-dependent persistence engine for studying collapse under constrained graph evolution. The system evolves seeded interaction graphs through a frozen hierarchy, measures persistence and recovery at each level, and identifies where structure fails.

This release establishes a validated base regime:

- collapse concentrates at levels `2-3`
- the strongest pre-collapse predictor is `delta_persistence(1->2)`
- held-out prediction accuracy is `0.875`
- the transition threshold is narrowly localized near `-0.1176`
- the deciding mechanism is consolidation failure within connected support, not fragmentation

Control result:

- schedule shift acts as a control parameter
- increasing shift advances the critical transition step
- beyond the base regime, the original `2 vs 3` split is compressed and then erased

This is a narrow, reproducible, regime-local claim: collapse is predictable before the boundary, and the mechanism is operationally grounded in the validated base regime.
